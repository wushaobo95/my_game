## 法宝系统重构完成

1. **配置更新** (`core/game-config.js`)
   - 添加统一法宝物品配置：图标"?", 颜色"#ffcc00"

2. **文案更新** (`core/strings.js`)
   - 添加法宝选择界面文案：标题"选择法宝"，描述"从以下法宝中选择一个"

3. **ItemPickup类重构** (`js/entities/gem.js`)

   - 统一法宝物品拾取时触发选择界面
   - 统一法宝物品显示特殊图标和颜色

   - 法宝池非空时返回统一法宝物品标记`{isUnified: true}`
   - 法宝池耗尽时返回`null`

5. **showItemChoiceScreen函数** (`js/systems/upgrade.js`)
   - 显示法宝选择界面，从可用法宝池随机选择最多3个法宝
   - 玩家必须选择一个法宝，不能跳过
   - 选择后应用法宝效果，并从池子中移除（不放回抽取）

### 测试步骤

1. **启动游戏**
   - 在浏览器中打开 `index.html`
   - 开始新游戏

2. **测试统一法宝物品掉落**
   - 击败第一个Boss（约30秒后出现）
   - Boss死亡时应掉落一个显示"?"的统一法宝物品

3. **测试法宝选择界面**
   - 拾取统一法宝物品
   - 游戏暂停，显示法宝选择界面
   - 应显示3个法宝选项（如果可用法宝不足3个，显示实际数量）

4. **测试法宝选择**
   - 点击一个法宝选项
   - 法宝效果应立即生效
   - 游戏继续

5. **测试不放回抽取**
   - 再次击败Boss，拾取法宝
   - 选择界面不应显示已选择的法宝

6. **测试法宝池耗尽**
   - 选择所有8个法宝后
   - Boss死亡时不再掉落法宝物品

- 玩家必须选择一个法宝，不能跳过
