# 卡皮巴拉冒险 - 资源规格文档

## 概述
本文档定义了游戏《卡皮巴拉冒险》所需的所有视觉和音频资源规格。所有资源应遵循以下标准：

## 文件命名规范
- 使用小写字母和下划线
- 格式：`[类别]_[名称].[扩展名]`
- 示例：`player_normal.png`, `enemy_boss.png`

## 目录结构
```
assets/
├── sprites/
│   ├── player/          # 玩家相关精灵图
│   ├── enemies/         # 敌人精灵图
│   ├── bullets/         # 子弹和特效
│   ├── items/           # 宝石、道具
│   ├── effects/         # 粒子和特效
│   └── background/      # 背景和UI元素
├── audio/
│   ├── sfx/             # 音效文件
│   └── music/           # 背景音乐
└── fonts/               # 自定义字体（可选）
```

## 技术规格

### 图像规格
- **颜色模式**：RGBA（带透明度）
- **位深度**：32位
- **压缩**：PNG-24（无损）
- **DPI**：72（标准屏幕分辨率）
- **抗锯齿**：启用
- **透明背景**：所有精灵图必须使用透明背景

### 音频规格
- **采样率**：44.1kHz
- **位深度**：16位
- **声道**：立体声（音乐），单声道（音效）
- **比特率**：128-192 kbps（MP3）
- **标准化**：所有音效应标准化到相同响度

### 文件大小限制
- **精灵图**：< 100KB 每个
- **音效**：< 200KB 每个
- **背景音乐**：< 5MB 每个

## 创建指南

### 1. 精灵图创建流程
1. 根据规格创建图像
2. 确保透明背景
3. 保存为PNG格式
4. 放置在对应文件夹

### 2. 音频创建流程
1. 根据规格录制/合成音频
2. 标准化音量
3. 导出为MP3或OGG
4. 放置在对应文件夹

### 3. 测试资源
1. 将资源放入 `assets/` 文件夹
2. 启动游戏
3. 检查资源是否正确加载
4. 调整大小、颜色、音量等

## 默认回退机制

游戏实现了资源回退机制：
1. 如果资源文件不存在，游戏使用原有的Canvas绘制和Web Audio合成
2. 这允许逐步替换资源，无需一次性完成所有资源
3. 资源加载器会记录加载状态，便于调试
