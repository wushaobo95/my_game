## Context

《弧光幸存者》是一款吸血鬼幸存者风格的HTML5 Canvas游戏。玩家在无限生存模式中面对不断增强的敌人波次，通过升级和获取法宝来强化角色。

当前数值存在两个核心问题：
1. 前期5分钟难度偏大，敌人生成太快
2. 后期Boss被秒杀，玩家成长曲线与敌人难度曲线不匹配

本次调整只修改 `js/core/game-config.js` 中的数值，不改变任何游戏逻辑或功能行为。

## Goals / Non-Goals

**Goals:**
- 降低前期（0-5分钟）敌人压力，让玩家更容易上手
- 提高后期（5分钟+）Boss挑战性，防止被秒杀
- 保持爽快割草体验，让玩家成长曲线与敌人难度曲线更匹配

**Non-Goals:**
- 不新增任何游戏功能或机制
- 不改变游戏逻辑代码
- 不调整视觉效果或音效

## Decisions

### 决策1：降低玩家攻击力成长
- **选择**: 从每次升级+8降低到+5
- **原因**: 玩家攻击力成长过高是后期秒Boss的主要原因。+8的成长导致10分钟时攻击力达到174+，DPS超过800
- **备选方案**: 
  - 提高Boss HP更多 → 会导致前期Boss太难
  - 降低攻速成长 → 会影响游戏手感

### 决策2：降低普通敌人HP成长
- **选择**: HP_SCALE从12降到10，HP_SCALE_PER_LEVEL从0.08降到0.06
- **原因**: 降低前期敌人HP，让玩家更容易击杀，建立正反馈
- **备选方案**:
  - 提高玩家初始攻击力 → 会影响游戏平衡
  - 降低敌人数量 → 会减少爽快感

### 决策3：大幅提高Boss HP
- **选择**: HP_SCALE从200提高到500，BOSS_HP_SCALE_PER_LEVEL从0.10提高到0.15
- **原因**: Boss是后期核心挑战，需要足够血量来考验玩家build
- **备选方案**:
  - 增加Boss技能 → 改动太大，超出本次范围
  - 降低玩家暴击率 → 会影响爽快感

### 决策4：降低敌人生成压力
- **选择**: BASE_RATE从2.0降到1.5，MAX_RATE从6降到5
- **原因**: 前期生成太快是难度大的主要原因，6只/秒让新手玩家难以招架
- **备选方案**:
  - 提高玩家移动速度 → 不能解决核心问题
  - 降低敌人伤害 → 不能解决清不完的问题

## Risks / Trade-offs

- **风险**: 数值调整可能过度，导致前期太简单或后期太难
  - **缓解**: 先小幅调整，根据测试反馈再微调

- **风险**: 降低生成率可能让游戏节奏变慢
  - **缓解**: 保持批量生成机制，确保后期仍有足够敌人

- **权衡**: 降低攻击力成长会影响玩家爽快感
  - **接受**: 通过保持多重射击、暴击等其他成长维度来弥补
