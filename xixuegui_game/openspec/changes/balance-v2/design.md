## Context

上一轮数值调整（balance-numbers）后，游戏前期体验改善，但后期Boss战太弱。测试发现：
- 第2个Boss开始，Boss被快速秒杀
- 多重射击是主要原因：3个额外投射物 = 4倍DPS，且无伤害衰减

本次调整目标：让后期Boss战有挑战性，同时保持爽快感。

## Goals / Non-Goals

**Goals:**
- 让Boss战持续8-15秒（而不是1-2秒）
- 通过伤害衰减机制平衡多重射击
- 适当提高敌人威胁度

**Non-Goals:**
- 不改变多重射击机制本身
- 不新增游戏功能
- 不大幅改变前期体验

## Decisions

### 决策1：添加伤害衰减
- **选择**: PROJECTILE_DAMAGE_DECAY = 0.65
- **原因**: 直接限制多重射击的DPS爆炸，每个额外投射物伤害为前一个的65%
- **计算**: 3个额外投射物时，总伤害倍数 = 1 + 0.65 + 0.42 + 0.27 = 2.34
- **备选方案**: 
  - 降低多重射�上限 → 会影响爽快感
  - 增加冷却时间 → 会降低游戏节奏

### 决策2：大幅提高Boss HP
- **选择**: HP_SCALE = 1500, BOSS_HP_SCALE_PER_LEVEL = 0.20
- **原因**: 考虑伤害衰减后，Boss需要足够血量来维持战斗时长
- **计算**: 20级Boss HP ≈ 6360，DPS ≈ 1216，击杀时间 ≈ 5秒
- **备选方案**: 
  - 只提高HP_SCALE → 等级成长不够
  - 只提高等级成长 → 基础值不够

### 决策3：提高敌人伤害
- **选择**: DAMAGE_SCALE_PER_LEVEL = 0.04, BOSS_DAMAGE_SCALE_PER_LEVEL = 0.05
- **原因**: 增加后期敌人威胁度，让玩家不能无脑站撸
- **备选方案**: 
  - 增加敌人数量 → 会影响性能
  - 增加新敌人类型 → 改动太大

## Risks / Trade-offs

- **风险**: 伤害衰减可能让多重射击感觉太弱
  - **缓解**: 0.65的衰减率仍然让多重射击有价值，只是不再4倍DPS

- **风险**: Boss HP太高可能让战斗太拖沓
  - **缓解**: 5-8秒的战斗时长是合理的，不会太拖

- **权衡**: 提高敌人伤害会增加难度
  - **接受**: 这是需要的，后期应该有压力
